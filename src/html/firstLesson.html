Что такое HTML и как он передаётся

HTML это конструктор страниц кода (расшифровка HTML: "HyperText Markup Language" - "Язык ГиперТекстовой Разметки"). Страницы HTML передаются по протоколу HTTP (расшифровка HTTP: "HyperText Transfer Protocol" - "Протокол Передачи ГиперТекста"). Но протосол HTTP сам по себе небезопасен. Поэтому придумали такую схему: HTTP + TLS ("Transport Layer Security" - "Защитный Слой Передачей") = HTTPS, т. е. добавили и протокол защитного слоя, и букву "S" (от слова "Security" - безопасность).

Теги HTML

Теги которые вы должны знать:

<! --> — коментарий (игнорируется браузером);

<a> </a> — ссылка;

<body> </body> — «тело» документа (обязательный тег!);

<div> </div> — блочный элемент;

<head> </head> — «голова» документа (обязательный тег!);

<html> </html> — документ (обязательный тег!);

<!DOCTYPE html> — тип документа (обязательный тег!);

<p> </p> — параграф;

<span> </span> — неблоковый элемент для стилей;

<! --> — комментарий от разработчиков. <!DOCTYPE html> — определяет тип документа (обязательный тег для страниц HTML!). <a> </a> — ссылочный элемент — ссылка (напр.: на скачивание, на рикролл, на видео и пр.). <body> </body> — «тело» документа или область контента документа (обязательный тег для страниц HTML!). <head> </head> — параметры документа такие как подключение стилей и скриптов, изменение кодировки и пр. (обязательный тег для страниц HTML!). <html> </html> — документ — всё, что внутри него считается документом (обязательный тег для страниц HTML!). <p> </p> — параграф — внутри него находится текст. Блочный элемент. <span> </span> — неблочны элемент стиля — внутри него находится текст, который впоследсвии стилизуется.
