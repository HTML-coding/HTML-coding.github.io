Блочные теги

Есть так назызваемые блочные теги. Такими называют div, button, a, p, img и пр.

Тег div полезен так и для JavaScript, так и для CSS. Тег button — без тега a — обычная кнопка, никак не функционирующая. Кстати, у каждого элемента HTML могут быть аргумены class="" или id="".
